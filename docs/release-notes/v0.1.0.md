# Release Notes v0.1.0-mvp

**リリース日**: 2026-01-09

## 概要

Washu App の初回 MVP リリースです。日本酒との出会いをサポートする PWA として、検索・詳細・最近見た・お気に入り機能をオフラインでも利用可能な形で提供します。

---

## 新機能

### フロントエンド (PWA)
- **検索 `/search`**: キーワード・地域フィルタによる銘柄検索。無限スクロール対応。
- **詳細 `/sake/:id`**: 銘柄の詳細情報（蔵元、地域、味わいタグ等）を表示。閲覧履歴を自動保存。
- **最近見た `/recent`**: 最大50件の閲覧履歴をオフラインでも表示。
- **お気に入り `/favorites`**: ★トグルで登録・削除。IndexedDB に永続化しオフラインアクセス可能。
- **オフライン対応**: Service Worker による静的アセットキャッシュ、`offline.html` フォールバック。
- **PWA インストール**: iOS Safari / Android Chrome / Desktop Chrome で A2HS 対応。

### バックエンド (FastAPI)
- **REST API**: `/api/v1/sake/search`, `/api/v1/sake/{id}`, `/api/v1/meta/regions`
- **PostgreSQL**: 銘柄データの永続化、デモデータ seed 機能
- **Health Check**: `/health` エンドポイント

### CI/CD
- **GitHub Actions**: backend / frontend テスト、Lighthouse CI、Playwright E2E
- **Docker Compose**: 開発環境の統一化

---

## 改善・整備 (Phase 6)

- **E2E 安定化**: IndexedDB の favorites を deterministic に seed（`seedFavoritesInPage` / `waitForFavoriteInIdb`）。
- **スクリーンショット**: SPA 経由で遷移し、PWA オフラインフォールバックを回避。
- **Lighthouse CI**: preview readiness を待機し、Lighthouse v12 に合わせて PWA カテゴリを無効化。
- **SEO**: `meta description` と `robots.txt` を追加。
- **Docs**: 文書作成ルールを追加（`docs/DOCS_MASTER_RULES.md`）。
